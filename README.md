# Cybersecurity Blog & Pentesting Writeups

This repository contains the source code and content for my GitHub Pages website, where I document my cybersecurity learning journey, penetration testing experiences, and hands-on practical research.

The platform is focused on sharing detailed walkthroughs, technical blogs, Capture The Flag (CTF) solutions, vulnerability analysis, and notes related to offensive security and ethical hacking.

## Contents

The repository includes content related to:

* Penetration Testing Walkthroughs
* TryHackMe Writeups
* Hack The Box Writeups
* VulnHub Machines
* Web Application Security
* Network Enumeration & Exploitation
* Privilege Escalation Techniques
* Vulnerability Assessment
* Red Teaming Concepts
* Cybersecurity Learning Notes
* Security Tools & Methodologies

## Purpose

The primary objective of this project is to:

* Document practical cybersecurity knowledge
* Share real-world pentesting methodologies
* Help beginners understand offensive security concepts
* Maintain technical notes and references
* Build a public knowledge base for continuous learning

## Technologies Used

This website is built using:

* GitHub Pages
* Jekyll
* Markdown
* HTML/CSS
* YAML Configuration
